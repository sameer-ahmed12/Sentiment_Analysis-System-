# Sentiment_Analysis-System-
Sentiment Analysis Tool — Description

This is a Command-Line Sentiment Analysis Tool built using machine learning with the scikit-learn library.

The script trains multiple models (Naive Bayes and Logistic Regression) on a small labeled dataset of text (positive, negative, neutral). It then allows users to input any sentence in real-time and predicts its sentiment.

🔹 Features:
Text preprocessing (cleaning, normalization)
Multiple ML models:
Naive Bayes (TF-IDF & CountVectorizer)
Logistic Regression (TF-IDF)
Accuracy & F1-score evaluation
Interactive CLI (Command Line Interface)
Real-time sentiment prediction
Confidence score + probability breakdown

⚙️ Installation & Setup
First, install the required library:
pip install scikit-learn

▶️ How to Run
Run the script in your terminal / CLI:
python sentiment_analysis.py
